# Atte

企業の勤怠管理システム
![index](https://user-images.githubusercontent.com/112733879/216904092-37309a04-a394-47a0-add9-9096bf676298.png)

## 作成した目的

人事評価のため

## アプリケーション URL

http://localhost:8000/login

## 機能一覧

-   会員登録
-   ログイン
-   ログアウト
-   勤務開始
-   勤務終了
-   休憩開始
-   休憩終了
-   日付別勤怠情報取得
-   ページネーション

## 使用技術（実行環境）

-   PHP 　 8.1.6
-   Laravel 　 8.83.27
